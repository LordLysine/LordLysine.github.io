### 使用步骤
1. 将以上文件按目录结构放入你的仓库
2. 运行 `bundle exec jekyll serve` 本地预览
3. 推送到 GitHub 仓库的 `main` 分支
4. 在仓库设置中启用 GitHub Pages

- 首页自动展示最新 5 篇文章摘要
- 支持 Markdown 语法 + 代码高亮
- 自动生成文章导航（上一篇/下一篇）
